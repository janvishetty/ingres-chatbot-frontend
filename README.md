# INGRES Chatbot Frontend

A responsive chatbot interface for visualizing groundwater data such as recharge trends, extraction rates, and groundwater stages. Built using React (via CDN), Chart.js, and modern UI styling.

---

## Features

* Interactive chatbot interface
* Dynamic data visualization using charts (Chart.js)
* Table-based structured responses
* Multi-language selection support (UI level)
* Lightweight frontend prototype without build tools
* Responsive and clean design

---

## Tech Stack

* React (CDN) – UI rendering
* Chart.js – Data visualization
* HTML5 and CSS3 – Layout and styling
* JavaScript (ES6) – Application logic

---

## How to Run Locally

1. Clone the repository:

``
git clone https://github.com/janvishetty/ingres-chatbot-frontend.git
``

2. Navigate into the project folder:

``
cd ingres-chatbot-frontend
``

3. Open the application:

``
open index.html
``
